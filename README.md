# To-Do List App ✅ (Modernized)

A simple, beautiful, and interactive To-Do List application built with HTML, CSS, and vanilla JavaScript. This project allows users to add tasks, mark them as complete, and delete them. It also features data persistence, meaning your tasks are saved in the browser and won't disappear when you refresh the page!

![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/javascript-%23323330.svg?style=for-the-badge&logo=javascript&logoColor=%23F7DF1E)

## ✨ Features

* **Add Tasks:** Quickly add new tasks to your list using the input field, now supporting the `Enter` key!
* **Mark as Complete:** Click on any task to cross it out and mark it as done.
* **Delete Tasks:** Remove unwanted or completed tasks by clicking the "×" icon, now with a sleek fade-out animation.
* **Local Storage:** Tasks are automatically saved to your browser's local storage. You can safely close or refresh your browser, and your list will still be there when you come back!
* **Modernized UI:** Clean and modern glassmorphism UI with a beautiful gradient background, Google Fonts ('Outfit'), hover states, and smooth CSS transitions.

## 🛠️ Built With

* **HTML5** - Semantic structure for the app.
* **CSS3** - For styling, flexbox layouts, glassmorphism, CSS variables, and animations.
* **JavaScript (Vanilla)** - For the logic, DOM manipulation, event delegation, and local storage.

## 🧠 What I Learned

Building this project helped me practice and understand:
* **DOM Manipulation:** Creating and removing HTML elements (like `<li>` and `<span>`) dynamically using JavaScript.
* **Event Delegation:** Efficiently managing click events on the task list to handle checking off tasks and deleting them.
* **CSS Pseudo-elements:** Using `::before` to create custom unchecked and checked circle icons.
* **CSS Animations & Transitions:** Adding sleek hover effects and keyframe animations for a premium feel.
* **Browser Local Storage:** Using `localStorage.setItem()` and `localStorage.getItem()` to save user data so it persists across page reloads.
